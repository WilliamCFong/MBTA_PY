============
Installation
============


Stable release
--------------

To install MBTA Python Wrapper, run this command in your terminal:

.. code-block:: console

    $ pip install mbta_py

This is the preferred method to install MBTA Python Wrapper, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for MBTA Python Wrapper can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/trigition/mbta_py

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/trigition/mbta_py/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/trigition/mbta_py
.. _tarball: https://github.com/trigition/mbta_py/tarball/master
